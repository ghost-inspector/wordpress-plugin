Display your latest Ghost Inspector test results for a single suite, on your dashboard. Ghost Inspector is a 3rd party service which allows you to build or record automated website tests in your browser and run them continuously from the cloud, monitoring your website for issues. In order to use this plugin, you must obtain a Ghost Inspector account at https://ghostinspector.com. This plugin will not send any data from your WordPress instance to the Ghost Inspector service, with the exception of the Ghost Inspector API key and Suite ID that you provide in the plugin's settings. This plugin will make requests from the server hosting your Wordpress instance to https://api.ghostinspector.com using the supplied API key and Suite ID to fetch data from your Ghost Inspector account and display it. For Ghost Inspector's complete Terms & Conditions, please visit https://ghostinspector.com/terms/.
