Display the latest test results of a single suite on your dashboard. To get started: sign up for an account on ghostinspector.com and add your API key and your suite ID to your Ghost Inspector settings here in Wordpress. Build or record automated website tests in your browser with Ghost Inspector. We run them continuously from the cloud, monitoring your website for issues.
